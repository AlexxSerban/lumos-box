# Images Folder

Place your images here for use in the website.

## Image Files Needed

### Hero Section
- `hero-background.jpg` - Background image for the hero section (recommended: 1920x1080px or larger)

### Gallery Section
- `gallery-1.jpg` - First gallery image (recommended: 800x600px or larger)
- `gallery-2.jpg` - Second gallery image
- `gallery-3.jpg` - Third gallery image
- `gallery-4.jpg` - Fourth gallery image

## How to Use

1. Add your image files to this folder
2. Images in the `public` folder can be referenced directly in your code using:
   - `/images/your-image-name.jpg`
3. The images will be automatically available when you run the development server

## Image Recommendations

- **Format**: JPG or PNG
- **Hero Background**: High resolution (1920px width minimum) for best quality
- **Gallery Images**: Aspect ratio 4:3 works best (800x600px or similar)
- **File Size**: Optimize images for web to ensure fast loading times
